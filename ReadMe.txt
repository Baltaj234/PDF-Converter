This is the free PDF Converter
Be able to seamlessy drag and drop files into the converter and get
a full fledged PDF.

Required node packages to install to run this program:
npm install express multer pdf-lib mammoth fs
Also need mySQL database configured and installed on your operating system, 
Create a database as specified in the server.js file and create the tables via command line in the
tables.sql file.
After the use node server.js to run the file from command line, and it should run on LocalHost 3000.
